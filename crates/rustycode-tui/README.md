# RustyCode TUI

Terminal user interface for RustyCode with interactive prompts, message rendering, and status display.

## Features

- **Interactive Prompts**: Confirm, Input, Select, MultiSelect widgets
- **Message Rendering**: Render AI messages with tool calls and streaming
- **Status Bar**: Display provider, model, token usage
- **File Finder**: Quick file navigation
- **Sidebar**: Session management and navigation
