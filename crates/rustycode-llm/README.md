# RustyCode LLM

LLM provider integrations for RustyCode with support for Anthropic, OpenAI, OpenRouter, and more.

## Features

- **Multi-Provider**: Anthropic, OpenAI, OpenRouter, Google Gemini, Ollama
- **Streaming**: Stream responses for real-time feedback
- **Tool Calling**: Structured tool/function calling support
- **Token Tracking**: Monitor token usage and costs

## Providers

- `anthropic`: Anthropic Claude models
- `openai`: OpenAI GPT models
- `openrouter`: OpenRouter aggregated models
- `google`: Google Gemini models
- `ollama`: Local Ollama models
